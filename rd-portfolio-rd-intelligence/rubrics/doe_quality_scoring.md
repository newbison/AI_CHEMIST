# DOE Quality Scoring Rubric

Assess whether a proposed experiment is decision-grade.

## Required Elements

- Clear hypothesis
- Evidence basis
- Defined X factors
- Defined factor ranges
- Defined Y responses
- Test methods
- Controls
- Replicates
- Randomization
- Blocking, if needed
- Decision rule
- Risk review gates

## Score

```text
High — ready for lab review
Medium — useful concept but needs DOE refinement
Low — exploratory idea, not execution-ready
```

## Common Failure Modes

- No decision rule
- Too many factors for available resources
- Missing controls
- No replicate plan
- Output Ys not linked to CTQs
- Test method not available
- IP/regulatory/manufacturing risks ignored
